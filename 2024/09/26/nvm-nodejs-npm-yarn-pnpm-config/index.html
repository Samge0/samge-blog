前言

npm、yarn、pnpm默认的cache目录在系统盘，现希望迁移其缓存cache及global目录到F:\.cache\nodejs下。

这里默认用windows安装为例

推荐路线：安装nvm -》 用nvm安装nodejs -》 配置npm -》【可选】安装yarn/pnpm（这两个也是跟npm类似的包管理工具，可选）
