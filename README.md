### butterfly主题需要额外安装：
```shell
npm install hexo-renderer-pug hexo-renderer-stylus
```

除了git拉趣源码的方式，还可以：
```shell
npm i hexo-theme-butterfly
```

增加搜索功能:
```shell
npm install hexo-generator-search --save

hexo new page search
```
